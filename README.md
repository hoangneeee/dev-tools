# Dev-tools

## Feature

### Convert type string
Using cmd `cmd+ctrl+c` for Mac

- Convert string slug to camel case
`dev-tools => devTools`

- Convert string camel case to snake
`devTools => dev_tools`

- Convert string camel case to slug
`dev_tools => dev-tools`

Convert to uppercase
`dev_tools => DEV_TOOLS`

Convert to lowercase
`DEV_TOOLS => dev_tools`
